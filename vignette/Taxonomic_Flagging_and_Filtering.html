Taxonomic Data

The taxonomic part of the biodeiversity data deals with the information related to the origin, habitat, chronology and other native information related to specific species. It majorly includes fields like class, kingdom, phylum, genus etc.

Functions Description and Methodology

The file Taxonomic_Flagging_and_Filtering contains functions which help in flagging and filtering data based on taxonnomic fields o fthe data.

The File contains three functions to preform the task.

Function 1:Wrapper for flagging data-Taxonomic_Flagging Simple function to call the main function for flagging data and returning the flagged data frame

Function 2:Wrapper for filtering the flag data-Taxonomic_Filtering Simple function to call the main function for filtering the data returning the filtererd data frame

Function 3:Main function:Taxonomic_Flagging_and_Filtering_main This function checks whether the input flag is valid and if so it then flags the data based on the flags and filters the data based on the flag, it returns both flaggedand filtered data frame depending on the user. Now this function only contains simple flagging anf filtering mechanism based on NA values.
